6 分类

  • char 转 String：String.valueOf('a') 或 Character.toString('a')
  • String 转 char（单个字符）："abc".charAt(0)（返回 'a'）
  • String 转 char[]："abc".toCharArray()（返回字符数组）

String 类：

String.valueOf(t.charAt(i)) 是 Java 中将 char 类型转换为 String 类型的常用方法，作用是把一个字符转换为对应的字符串对象。

  1. 长度与判断

    • length()：返回字符串长度
    • isEmpty()：判断是否为空字符串

  2. 字符访问

    • charAt(int index)：返回指定索引位置的字符
    • toCharArray()：转为字符数组

  3. 截取与分割

    • substring(int start)：从 start 截取到末尾
    • substring(int start, int end)：截取 [start, end) 区间
    • split(String regex)：按正则分割为字符串数组

  4. 查找与比较

    • indexOf(String str)：查找子串首次出现位置
    • lastIndexOf(String str)：查找子串最后出现位置
    • equals(Object obj)：比较内容是否相等
    • equalsIgnoreCase(String str)：忽略大小写比较

  5. 转换

    • toLowerCase()：转小写
    • toUpperCase()：转大写
    • trim()：去除首尾空格（Java 11+ 可用 strip() 更完善）

  6. 其他常用

    • contains(CharSequence s)：是否包含指定序列
    • startsWith(String prefix)：是否以指定前缀开头
    • endsWith(String suffix)：是否以指定后缀结尾
    • replace(char old, char new)：替换字符
    • valueOf(任意类型)：静态方法，将其他类型转为字符串

动态数组的基本 API

class MyArrayList {

// 向数组末尾添加元素，时间复杂度 O(1)（无扩容时），O(n)（需扩容时，需复制数组）

void add(E e);

// 向指定索引位置插入元素，后续元素自动后移，时间复杂度 O(n)（需移动索引后元素）

void add(int index, E e);

// 删除指定索引位置的元素，返回被删除元素，后续元素自动前移，时间复杂度 O(n)

E remove(int index);

// 删除第一个与目标元素相等的元素，删除成功返回 true，否则返回 false，时间复杂度 O(n)

boolean remove(E e);

// 获取指定索引位置的元素，时间复杂度 O(1)（数组随机访问特性）

E get(int index);

// 替换指定索引位置的元素，返回被替换的旧元素，时间复杂度 O(1)

E set(int index, E e);

// 判断数组是否包含目标元素，包含返回 true，否则返回 false，时间复杂度 O(n)

boolean contains(E e);

// 返回动态数组中元素的总个数，时间复杂度 O(1)

int size();

// 判断动态数组是否为空（元素个数为 0），返回 true 或 false，时间复杂度 O(1)

boolean isEmpty();

// 清空数组中所有元素（重置元素个数，不改变底层数组容量），时间复杂度 O(1)

void clear();

// 将动态数组转换为指定类型的数组，时间复杂度 O(n)（需遍历复制元素）

E[] toArray(E[] a);

// 手动指定数组最小容量，避免后续频繁扩容，时间复杂度 O(1)（无需扩容时），O(n)（需扩容时）

void ensureCapacity(int minCapacity);

// 将数组容量压缩至与当前元素个数一致，节省内存，时间复杂度 O(n)（需复制元素到新数组）

void trimToSize();

}

动态数组

在Java中，动态数组的核心实现类是 java.util.ArrayList **，其API围绕动态扩容、元素增删改查等操作设计，核心方法如下：

  1. 构造方法（初始化）

  •  new ArrayList<>() ：创建空的ArrayList（初始容量10，扩容时自动翻倍）。

  •  new ArrayList<>(int initialCapacity) ：指定初始容量（避免频繁扩容，适合已知大致元素数量的场景）。

  •  new ArrayList<>(Collection<? extends E> c) ：通过其他集合（如List、Set）初始化。

  •  add(E e) ：尾插元素

  •  add(int index, E e) ：指定索引插入元素

  •  remove(int index) ：删除指定索引元素，返回被删元素

  •  remove(Object o) ：删除首个匹配元素，返回是否成功

  •  get(int index) ：获取指定索引元素

  •  set(int index, E e) ：替换指定索引元素，返回旧值

  •  size() ：返回元素个数

  •  isEmpty() ：判断是否为空

  •  contains(Object o) ：判断是否包含指定元素

Java中无专门的“双链表”接口，但 java.util.LinkedList 是底层基于双向链表的实现类（实现 List 和 Deque 接口），完全具备双链表的所有特性（头尾节点双向引用、双向遍历、头尾操作O(1)）。以下是其核心API（按功能分类，突出双链表特性）：

LinkedList核心API（双链表特性）

// 构造方法

  •  LinkedList<>() ：创建一个空的双向链表。

  • LinkedList<>(Collection<? extends E> c)：通过已有集合（如List、Set）初始化双向链表，元素顺序与集合一致。

class MyDoublyLinkedList {

// 向链表头部插入元素，时间复杂度 O(1)

void addFirst(E e);

// 向链表尾部插入元素，时间复杂度 O(1)

void addLast(E e);

// 向指定索引位置插入元素，时间复杂度 O(n)（需遍历到目标索引）

void add(int index, E e);

// 删除并返回链表头部元素，时间复杂度 O(1)

E removeFirst();

// 删除并返回链表尾部元素，时间复杂度 O(1)

E removeLast();

// 删除并返回指定索引位置的元素，时间复杂度 O(n)（需遍历到目标索引）

E remove(int index);

// 删除第一个与目标元素相等的节点，删除成功返回 true，否则返回 false，时间复杂度 O(n)

boolean remove(E e);

// 获取链表头部元素（不删除），时间复杂度 O(1)

E getFirst();

// 获取链表尾部元素（不删除），时间复杂度 O(1)

E getLast();

// 获取指定索引位置的元素，时间复杂度 O(n)（需遍历到目标索引）

E get(int index);

// 替换指定索引位置的元素，返回被替换的旧元素，时间复杂度 O(n)（需遍历到目标索引）

E set(int index, E e);

// 判断链表是否包含目标元素，包含返回 true，否则返回 false，时间复杂度 O(n)

boolean contains(E e);

// 返回链表中元素的总个数，时间复杂度 O(1)

int size();

// 判断链表是否为空（元素个数为 0），返回 true 或 false，时间复杂度 O(1)

boolean isEmpty();

// 清空链表中所有元素，时间复杂度 O(1)（仅需置空头尾节点与重置 size）

void clear();

}
